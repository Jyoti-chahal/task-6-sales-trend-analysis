# 📊 Task 6: Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume from the `online_sales_orders` table using SQL.

## What I Did
- Checked for missing or null values in key columns.
- Grouped data by year and month using SQL functions.
- Calculated:
- 📈 Total monthly revenue using `SUM(amount)`
- 📦 Monthly order volume using `COUNT(DISTINCT order_id)`
- Identified:
- 🏆 Highest revenue month
- 📉 Lowest order volume month

## Files Included
- `sales_trend_analysis.sql`: SQL queries with comments
- `online_sales_orders.csv`: Sample dataset used for analysis
- `screenshot.png`: Screenshot showing SQL queries with output
- `README.md`: This documentation file

## Outcome
Grouped sales data by time, applied aggregations, and extracted key business insights from trends.
